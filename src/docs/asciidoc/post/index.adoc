== 게시글 API

게시글 목록 조회는 성능 개선 단계를 비교 측정하기 위해 V1~V4 네 가지 버전으로 제공된다.

- V1 (`/api/v1/posts`): 인덱스만 건 원본 오프셋 페이징
- V2 (`/api/v2/posts`): 커버링 인덱스 + 지연 조인
- V3 (`/api/v3/posts`): V2 + 페이지 블록 상한 카운트
- V4 (`/api/v4/posts`): 날짜 앵커 + 커서 기반

=== 게시글 조회수 증가 (V1~V3)

게시글 조회수 증가는 동시성 제어 전략을 비교 측정하기 위해 V1~V3 세 가지 버전으로 제공된다.

- V1 (`/api/v1/posts/{postId}/view-count`): 낙관적 락 (@Version + 재시도)
- V2 (`/api/v2/posts/{postId}/view-count`): 비관적 락 (select for update)
- V3 (`/api/v3/posts/{postId}/view-count`): 원자적 UPDATE (운영 방식)
